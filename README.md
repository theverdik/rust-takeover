# Rust Takeover
A clicker game about rust ironically written in C

---

## How to run

```
$ git clone https://github.com/theverdik/rust-takeover.git
```
and
```
$ make deps release
```
or
```
$ make deps debug
```
then
```
$ ./build/release/rust-takeover
```
or
```
$ ./build/debug/rust-takeover
```
